## To get started:

<br />

Clone the repo

```sh
$ git clone https://github.com/shitpy/shitpy.git
$ cd shitpy
```
for python version: 2
```sh
$ python2
```

for python version: 3
```sh
$ python3
``` 

<br />

Try out your first code with shitpy! Happy Pooping :)
```
>>> from shitpy import int
>>> a = int(2)
>>> b = int(5)
>>> a + b
```
